## Deployment Steps

### Step 1: Deploy Blueprint

1. Go to https://render.com
2. Click **New +**
3. Select **Blueprint**
4. Connect your GitHub repository
5. Select `render.yaml` file
6. Click **Apply**

### Step 2: Configure Backend

1. In Render Dashboard, click `waey-backend`
2. Go to **Environment** tab
3. Add these variables:

```env
# Required
DATABASE_URL=mongodb+srv://username:password@cluster.mongodb.net/waey?retryWrites=true&w=majority

# Optional (Email notifications)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password

# Optional (AI Chatbot)
GEMINI_API_KEY=your-gemini-api-key
```

4. Click **Save Changes**

### Step 3: Monitor Deployment

1. Go to **Logs** tab
2. Wait for build to complete (~5 minutes)
3. Look for: "Server running on port 10000"
4. Check health: `https://waey-backend.onrender.com/health`

### Step 4: Verify Frontend

1. Go to `waey-frontend` service
2. Check **Logs** for build success
3. Access: `https://waey-frontend.onrender.com`
4. Test homepage loads

## Troubleshooting

### Backend Fails to Start

**Check:**
- [ ] MongoDB connection string is correct
- [ ] MongoDB user/password are correct
- [ ] Network access allows all IPs (`0.0.0.0/0`)
- [ ] DATABASE_URL env var is set

**Logs show:**
```
❌ Error: MongoServerError: Authentication failed
```
→ Fix: Check username/password in connection string

### Frontend Shows Blank Page

**Check:**
- [ ] Backend is running (check health endpoint)
- [ ] VITE_API_URL is set correctly
- [ ] Browser console for errors (F12)

**Wait 5-10 minutes** - backend takes longer to deploy

### Stories/Articles Not Showing

**Run manual seed:**
1. Render Dashboard → `waey-backend`
2. **Shell** tab → Connect
3. Run: `npm run seed`
4. Refresh frontend

## Environment Variables Reference

### Backend (render.yaml)

**Auto-generated:**
- `JWT_SECRET` - Generated by Render
- `REFRESH_TOKEN_SECRET` - Generated by Render
- `REDIS_URL` - From Redis service

**You must add:**
- `DATABASE_URL` - MongoDB connection string

**Optional:**
- `SMTP_HOST`, `SMTP_USER`, `SMTP_PASSWORD` - For emails
- `GEMINI_API_KEY` - For AI chatbot

### Frontend (render.yaml)

**Auto-generated:**
- `VITE_API_URL` - From backend service
